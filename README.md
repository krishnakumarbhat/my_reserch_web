# Krishnakumar B — Portfolio

A premium, static research portfolio website built with pure HTML, CSS, and JavaScript.

## 🚀 Features

- **Zero Dependencies** — No frameworks, no build tools, pure static files
- **GPU-Accelerated Animations** — Typing effect, scroll reveals, floating particles, counter animations
- **Dark Theme** — Premium dark design with indigo/violet accent colors
- **Fully Responsive** — Mobile-first design with hamburger nav
- **SEO Optimized** — Semantic HTML5, meta tags, Open Graph
- **Fast Loading** — No npm packages, minimal asset size

## 📂 Structure

```
├── index.html        # Main HTML (609 lines)
├── style.css         # Full CSS with animations (1,479 lines)
├── script.js         # Vanilla JS interactions (216 lines)
├── assets/
│   └── profile.png   # Profile photo
├── .nojekyll         # GitHub Pages config
└── README.md
```

## 🌐 Deploy on GitHub Pages

1. Push this repository to GitHub
2. Go to **Settings → Pages**
3. Set source to **Deploy from a branch → main → / (root)**
4. Your site will be live at `https://yourusername.github.io/repo-name/`

## 📊 Performance

- **Total size**: ~87KB (HTML + CSS + JS)
- **No external JS dependencies**
- **Font loading**: Google Fonts with `preconnect`
- **Image optimization**: `loading="eager"` for hero, lazy elsewhere
